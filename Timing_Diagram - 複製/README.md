# IC Waveform Generator (時序圖產生器)

這是一個專為數位電路設計（Digital IC Design）與驗證（Verification）開發的輕量級時序圖繪製工具。它允許使用者直觀地建立、編輯並匯出高品質的波形圖，適用於撰寫規格書（Spec）或溝通介面時序。

## ✨ 主要功能 (Features)

*   **多樣化訊號支援**：支援 `CLK` (時脈), `INPUT` (輸入), `OUTPUT` (輸出), `INOUT`, `BUS` (匯流排) 等多種訊號類型。
*   **直觀的波形編輯**：
    *   **點擊 (Click)**：快速切換單一位元訊號的高低電位 (0/1)。
    *   **拖曳繪製 (Paint)**：按住滑鼠左鍵拖曳，可連續設置一段區間的值。
    *   **匯流排編輯 (Bus Editor)**：雙擊或選取匯流排區塊，可輸入 Hex/Bin/Dec 數值，並支援自定義顏色與時序長度。
*   **靈活的時序控制**：
    *   支援在現有波形中 **插入 (Insert)** 或 **覆寫 (Overwrite)** 新的狀態。
    *   可拖曳波形區塊的邊緣來調整持續時間 (Duration)。
    *   **區塊移動 (Block Move)**：按住 `Ctrl` 並拖曳已選取的區塊，可將其移動到其他時間點。
*   **視覺化優化**：
    *   **Tech Theme**：內建現代化的深色霓虹風格介面 (Dark Mode + Neon)。
    *   **自定義顏色**：每個訊號或甚至每個狀態區塊都可以設定獨立的顏色 (`IDLE`, `READ`, `WRITE` 等)。
    *   **釘選功能 (Pin)**：常用的關鍵訊號 (如 `clk`, `rst`) 可釘選至頂部，程式重啟後自動載入。
*   **高品質匯出**：
    *   支援將波形圖匯出為 `.png`, `.jpg`, `.bmp` 圖片。
    *   匯出時可自定義背景色、字體顏色與大小，確保圖片符合文件需求。

## ⌨️ 快捷鍵與操作指南 (Shortcuts & Controls)

### 🖱️ 滑鼠操作 (Mouse Actions)

| 操作                | 作用範圍           | 功能描述                                                                          |
| :------------------ | :----------------- | :-------------------------------------------------------------------------------- |
| **左鍵點擊**        | 單一位元訊號 (0/1) | 切換訊號電位 (Toggle High/Low)。                                                  |
| **左鍵拖曳 (繪製)** | 單一位元訊號 (0/1) | 連續繪製高電位或低電位 (Paint functionality)。                                    |
| **左鍵點擊**        | 匯流排 (Bus)       | 選取該狀態區塊，右側面板會顯示詳細資訊供編輯。                                    |
| **左鍵拖曳 (邊緣)** | 任意波形區塊       | 調整該狀態的持續長度 (Extend/Shrink Duration)。                                   |
| **拖曳訊號名稱**    | 左側訊號列表       | 上下拖曳可重新排列訊號順序 (Reorder Signals)。                                    |
| **Ctrl + 滾輪**     | 繪圖區 (Canvas)    | **縮放波形寬度 (Zoom Width)**。                                                   |
| **Ctrl + 左鍵拖曳** | 已選取區塊         | **移動區塊 (Move Block)**：將選取的波形區塊移動至新的時間點 (支援插入/覆寫模式)。 |

### ⌨️ 鍵盤操作 (Keyboard Shortcuts)

| 按鍵           | 功能描述                                               |
| :------------- | :----------------------------------------------------- |
| **⬅️ 左方向鍵** | 將選取框移動至上一個邏輯轉態點 (Previous Transition)。 |
| **➡️ 右方向鍵** | 將選取框移動至下一個邏輯轉態點 (Next Transition)。     |

### ⚙️ 其他操作細節

1.  **新增訊號**：點擊左側面板的 `Add` 按鈕，或設定好名稱與類型後點擊。
2.  **刪除訊號**：選取訊號後點擊 `Del` 按鈕。
3.  **釘選訊號 (Pin)**：點擊訊號名稱右側的圖釘圖示 (📌)。
    *   **亮青色**：已釘選 (重啟後保留)。
    *   **灰色**：未釘選。
4.  **編輯模式 (Edit Mode)**：
    *   在右方編輯面板中，可選擇 `Overwrite` (直接覆蓋原有數值) 或 `Insert` (插入新數值並推擠後方波形)。
